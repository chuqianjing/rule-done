# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

---

## [1.0.0] - 2026-03-26

### 新增

- **双角色系统**：支持「党支部管理员」和「发展成员」两种角色模式
- **基本信息管理**
  - 管理员端：支部信息、上级党委信息、公共字段配置
  - 成员端：个人基本信息填写与保存
- **模板管理**
  - 内置 17 种标准入党材料模板
  - 支持模板专有字段配置
  - 成员端材料锁定功能
- **文档生成**
  - 基于 Word 模板的占位符自动替换
  - 支持单个和批量导出
  - 智能字段映射（如「出生年月」→「出生日期」）
- **配置同步**
  - 支持管理员配置导出/导入
  - 支持通过 URL 自动同步配置
  - 启动时自动检查配置更新
- **安全特性**
  - 基于 Argon2 + AES-GCM 的数据加密
  - 密码设置与验证
  - 加密数据存储
- **用户界面**
  - 现代化 UI 设计（基于 PyQtDarkTheme）
  - 侧边导航栏布局
  - 响应式表单组件

### 技术栈

- GUI: PySide6
- Word 处理: python-docx + docxtpl
- 加密: cryptography + argon2-cffi
- 网络: requests
