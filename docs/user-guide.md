## 快速入门（5 分钟上手）

1. **安装应用**

   - 前往 [Releases 页面](../../releases) ，根据操作系统下载最新版本
      - Windows：`入档_vX.X.X_windows.zip`
      - macOS：`入档_vX.X.X_macos.zip`
   - 解压后双击运行即可

2. **首次启动时选择角色**

   - 选择「党支部管理员」：配置支部信息、模板通用字段
   - 选择「发展成员」：填写个人信息、生成材料

3. **基本工作流程**

   ```
   管理员填写配置 → 导出配置 → 成员导入配置 → 填写个人信息 → 生成材料
   ```

## 党支部管理员操作指南

### 填写配置

#### 1. 填写支部基本信息

进入「基本信息」页面，根据页面内容和实际情况填写各项信息。点击「保存」按钮保存配置。

>注：为便于管理员将更多发展党员工作的重要信息统一汇总，本工具在当前页面设置了对应板块，以供管理员按需使用。

#### 2. 配置模板通用字段

进入「模板列表」页面，点击需要配置的模板，进入「模板详情」页面配置当前发展批次成员在该模板的通用字段。

这些通用字段会自动应用到目标成员生成的材料中，例如：
- 支委会意见
- 投票人数
- 日期落款

本工具为模板字段的配置提供了如下三种方式，管理员可按需使用：
- 锁定：勾选锁定框，管理员设定的该字段值在成员端会固定显示（不论该字段是否为空值），成员无法修改
- 提示：管理员为该字段填写相应的值，但不勾选锁定框，该字段值在成员端会以提示的方式呈现，成员可根据需要修改其值
- 无：管理员不配置该字段（既不填写值、也不勾选锁定框），该字段在成员端无任何配置信息，成员根据个人情况来填写

此外，针对需要手写的材料，本工具在其「模板详情」页面为管理员提供填写要求的录入板块。

>注：为更好提升工作效率，针对同批次成员的当前阶段各项流程，建议党支部管理员统筹安排、积极联系上下级组织，使每项程序仅进行一次，从而使特定模板通用字段可更多地适用于当前批次成员。

### 导出与分发配置

#### 1. 导出为本地文件

进入「系统设置」页面，点击「导出配置」按钮，选择保存位置，生成 `admin_config.json` 文件。

>注：如使用下述云端同步方式分发配置，请确保导出配置前，已在「基本信息」页面的「系统设置」分组中，填写「配置同步URL」。

#### 2. 配置云端同步

将导出的 `admin_config.json` 上传至可公开访问的网页静态资源并获取其 URL，可使用 GitHub Gist、云存储服务的直链、或自建服务器。成员端启动时将自动从该 URL 同步最新配置。

>注：强烈建议管理员使用云端同步方式分发配置，但若有特殊情况如网页无法访问时，亦可使用群聊等方式分发文件，或直接在成员端自行配置。

### 维护配置

#### 1. 锁定配置

设定好当前阶段的相关配置后，可在「系统设置」页面的「锁定管理」分组锁定所有配置，避免误触修改。锁定后如需修改，亦可解锁配置。

#### 2. 设置密码

若需避免他人查看或更改信息，可在「系统设置」页面的「密码管理」分组设置密码，设置后管理员端每次启动应用时需要输入密码验证。

## 发展成员操作指南

### 获取支部配置

首次使用时，需要获取所在支部的管理员配置：

#### 方式一：通过 URL 同步

- 向支部管理员获取配置同步 URL
- 启动应用时选择「通过 URL 同步配置」
- 输入 URL 完成同步

#### 方式二：从本地导入

- 向支部管理员获取 `admin_config.json` 文件
- 启动应用时选择「从本地导入配置文件」
- 选择配置文件完成导入

### 完善成员信息

#### 1. 填写基本信息

「基本信息」页面提供了「支部公共信息」和「个人基本信息」两个板块，成员可在前者中了解支部若干重要信息，可在后者中点击「编辑」填写个人信息，完成后点击「保存」。

#### 2. 填写模板字段

进入「模板列表」页面，点击需要填写的模板，在「模板详情」页面填写该模板的专有字段，点击「保存」。

管理员会根据需要设定若干字段的值，成员可根据字段填写栏后的信息有无来填写。

### 生成材料文档

#### 方式一：单个导出

在模板详情页面，点击「导出」按钮，系统自动生成填充好的 Word 文档。如有需要，可在「系统设置」页面中修改导出文件的保存位置。

#### 方式二：批量导出

在「模板列表」页面，灵活使用 Ctrl 或 Shift 键来勾选需要导出的模板。点击「批量导出」按钮即可生成相应文档。

### 维护党员档案

#### 1. 锁定材料

当某份材料填写完成，且不需要再修改时，可以通过锁定操作来固定留存该材料文件的信息。在「模板详情」页面填写完所有字段，点击「锁定材料」按钮，锁定后该材料将变为只读状态。

> 注：锁定操作不可完全撤销，请确保材料填写无误再锁定

#### 2. 密码保护

进入「系统设置」页面，在相应分组中设置密码保护个人档案数据，下次启动时需要输入密码。

#### 3. 数据迁移

成员如需在其他设备上操作，可提前在「系统设置」页面中将个人数据 `member_info.json` 文件导出并发送至目标设备，在该设备上导入个人数据即可。

## 高级功能

用户可根据需要按如下步骤在应用中添加并使用模板材料：

### 1.编辑 Word 模板：

- 准备 Word 模板文件（`.docx` 格式）
- 在模板中使用占位符语法：`{{ 字段名 }}`
- 将模板文件放入 `resources/templates/` 目录
- 重启应用后即可使用新模板

## 故障排除

### 常见问题

#### Q: 程序启动时提示"密码错误"

A: 请确认输入的密码正确。如果忘记密码，需要删除 `data/` 目录下的相关 JSON 文件重新开始。

#### Q: 导出的 Word 文档打开后格式错乱

A: 请确保使用 Microsoft Word 或 WPS Office 打开。部分在线文档工具可能不完全兼容。

#### Q: 配置同步失败

A: 请检查：
1. URL 是否正确且可访问
2. 网络连接是否正常
3. URL 返回的是否为有效的 JSON 格式

#### Q: 某些字段没有自动填充

A: 请检查：
1. 对应的基本信息是否已填写并保存
2. 模板中的占位符名称是否与字段名匹配

### 数据备份

建议定期备份 `data/` 目录下的文件：

- `admin_config.json` - 管理员配置
- `member_info.json` - 成员个人信息
- `system_settings.json` - 系统设置
