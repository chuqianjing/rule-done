## 六、模块间交互流程

### 6.1 应用启动流程
```
main.py
  ↓
MainWindow.__init__()
  ↓
PermissionController.detect_mode()
  ↓
根据模式加载对应页面
  - developer: AdminConfigPage
  - admin: AdminConfigPage
  - student: BasicInfoPage
```

### 6.2 数据保存流程
```
UI 层（BasicInfoPage）
  ↓
DataManager.save_student_data()
  ↓
StudentManager.save_data()
  ↓
Validators.validate_data()
  ↓
JSONStorage.write_json()
```

### 6.3 Word 生成流程
```
UI 层（ExportDialog）
  ↓
TemplateEngine.generate_document()
  ↓
TemplateManager.get_template()
  ↓
DataManager.merge_data_for_template()
  ↓
WordGenerator.generate()
  ↓
保存 Word 文档
```

## 七、错误处理与日志

### 7.1 异常处理策略
- 文件不存在：返回默认值或空数据结构
- 数据格式错误：记录错误，提示用户
- Word 生成失败：回滚，显示错误信息

### 7.2 日志记录
```python
import logging

logger = logging.getLogger(__name__)
logger.setLevel(logging.INFO)

# 记录关键操作
logger.info("保存学生数据")
logger.error("Word 生成失败", exc_info=True)
```
