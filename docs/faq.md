## 安装与启动

### Q: 支持哪些操作系统？

A: 支持 Windows 10/11、macOS 10.15+ 和主流 Linux 发行版（如 Ubuntu 20.04+）。

### Q: Python 版本有什么要求？

A: 需要 Python 3.10 或更高版本。推荐使用 Python 3.11。

### Q: 安装依赖时报错怎么办？

A: 尝试以下步骤：

```bash
# 1. 升级 pip
pip install --upgrade pip

# 2. 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 3. 如果 PySide6 安装失败，单独安装
pip install PySide6 --no-cache-dir
```

### Q: 首次启动时如何选择角色？

A: 首次启动会进入用户模式，弹窗询问以「党支部管理员」还是「发展成员」身份体验。选择后会记住你的选择。

## 管理员模式

### Q: 管理员需要配置哪些信息？

A: 主要包括：
- **支部信息**：支部名称、支部书记
- **上级党委信息**：党委名称、党委书记
- **公共字段**：学校名称、学院名称
- **模板通用字段**：培养联系人、介绍人等各模板的公共填写项

### Q: 如何将配置分发给成员？

A: 两种方式：

1. **本地文件**：在「系统设置」中导出配置文件，通过 U 盘、邮件等方式分发
2. **云端同步**：将配置文件上传到服务器，设置「配置同步URL」，成员端自动同步

### Q: 配置同步 URL 支持哪些服务？

A: 任何返回 JSON 内容的公开 URL 均可，例如：
- GitHub Gist（使用 Raw 链接）
- 云存储直链（如阿里云 OSS、腾讯云 COS）
- 自建服务器

## 成员模式

### Q: 如何获取支部配置？

A: 首次启动时会引导你选择：
- **从本地导入**：选择管理员提供的 JSON 配置文件
- **通过 URL 同步**：输入管理员提供的配置 URL

### Q: 需要填写哪些个人信息？

A: 基本信息包括：
- 姓名、性别、民族、出生日期、籍贯
- 身份证号、学历、学位、单位、职务
- 各时间节点（申请入党时间、确定积极分子时间等）

### Q: 什么是「锁定材料」？

A: 当某份材料填写完成且确认无误后，可以锁定该材料。锁定后：
- 该材料变为只读状态
- 无法再修改内容
- 防止误操作导致数据丢失

> 注意：锁定操作**不可撤销**！

### Q: 可以修改支部的公共字段吗？

A: 不可以。公共字段由管理员统一配置，成员只能查看不能修改。如需修改，请联系支部管理员。

## 模板与导出

### Q: 支持哪些材料模板？

A: 内置 17 种标准模板：
1. 入党申请书
2. 入党申请人谈话记实表
3. 入党积极分子备案表
4. 群团组织推优表
5. 入党积极分子培养考察表
6. 发展对象备案表
7. 发展对象政审表
8. 发展对象亲属函调表
9. 政治审查结论性材料
10. 发展对象短期集中培训考核鉴定表
11. 发展党员工作预审表
12. 入党志愿书
13. 预备党员备案表
14. 预备党员考察记实表
15. 转正申请书
16. 思想汇报
17. 其他需要存入的材料

### Q: 如何使用自定义模板？

A:
1. 准备 Word 模板（.docx 格式）
2. 使用 `{{ 字段名 }}` 作为占位符
3. 将文件命名为 `template_XXX_模板名称.docx`
4. 放入 `resources/templates/` 目录
5. 重启程序

### Q: 导出的文档格式是什么？

A: 导出的是 `.docx` 格式的 Word 文档，可用 Microsoft Word 或 WPS Office 打开编辑。

### Q: 可以批量导出所有材料吗？

A: 可以。在「模板列表」页面勾选需要导出的模板，点击「批量导出」即可。

## 数据与安全

### Q: 数据保存在哪里？

A: 保存在程序目录下的 `data/` 文件夹中：
- `admin_config.json` - 管理员配置
- `member_info.json` - 成员个人信息
- `system_settings.json` - 系统设置

### Q: 如何备份数据？

A: 复制 `data/` 目录下的所有 JSON 文件即可。建议定期备份。

### Q: 密码保护如何工作？

A:
- 设置密码后，数据会使用 AES-256 加密存储
- 每次启动程序需要输入密码验证
- 密码错误 3 次后程序自动退出

### Q: 忘记密码怎么办？

A: 很遗憾，由于数据已加密，无法恢复。唯一的办法是删除 `data/` 目录下的对应文件重新开始。

### Q: 数据会上传到服务器吗？

A: 不会。所有数据保存在本地，除非你主动配置了「配置同步URL」进行同步。

## 故障排除

### Q: 程序启动显示空白窗口

A: 可能原因：
1. Python 版本过低，请升级到 3.10+
2. PySide6 未正确安装，尝试重新安装

### Q: 导出的文档打开后乱码

A: 请使用 Microsoft Word 或 WPS Office 打开。部分在线文档工具可能不兼容。

### Q: 配置同步失败

A: 请检查：
1. URL 是否正确且可访问
2. 网络连接是否正常
3. URL 返回的是否为有效的 JSON

### Q: 某些字段没有自动填充

A: 请确认：
1. 对应的基本信息是否已填写并保存
2. 模板中的占位符名称是否正确

### Q: 如何完全重置程序？

A: 删除以下目录/文件后重新启动：
- `data/` 目录
- `exports/` 目录
