# 党员发展材料生成系统

## 项目简介

本地单机运行的桌面应用程序，用于规范化、自动化生成党员发展过程中所需的多份 Word 表格材料。

## 核心特性

- ✅ **完全离线运行**：不联网、不集中存储数据
- ✅ **数据本地化**：所有数据仅保存在本地
- ✅ **三级形态管理**：开发者态 → 管理员态 → 学生态
- ✅ **数据权限分离**：程序数据、管理员数据、学生数据严格区分
- ✅ **模板化生成**：基于 Word 模板自动生成合规文档

## 系统架构

详细架构设计请参考 [系统架构设计文档](docs/系统架构设计.md)

## 技术栈

- **桌面框架**：PyQt6
- **Word 处理**：python-docx / docxtpl
- **数据存储**：JSON 文件
- **编程语言**：Python 3.9+

## 快速开始

### 1. 环境要求

- Python 3.9 或更高版本
- Windows / macOS / Linux

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行程序

```bash
python main.py
```

## 使用流程

### 第一步：管理员配置（首次使用）

1. 启动程序
2. 进入管理员配置页面
3. 填写支部信息、上级党委信息等
4. 保存并锁定配置

### 第二步：学生使用

1. 在首页填写基本信息
2. 选择模板进行填写
3. 导出 Word 文档
